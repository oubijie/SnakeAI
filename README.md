# 贪吃蛇


### 项目介绍
试着用java开发一个简单的贪吃蛇程序。

### 项目部分
1. 核心算法
2. GUI显示部分
3. 控制部分
4. 主体程序部分

### 一些技术规范
1. 使用二维数组map来制作地图。
2. map地图数组大小可变。
3. 使用双缓冲技术刷新界面。
4. 使用LinkedList<Point>来实现蛇的主体。
5. 将GetKet实例传入Game中，来实现实时获取。
6. 将Game实例传入GameJpanel中，来实现实时绘制。
7. GetKey实例为独立线程来获取按键。
8. Game实例为独立线程用来自动运算游戏数据。
9. GameJPanel实例为独立线程，不断刷新，来实时显示游戏内容。

### 文件说明
1. Desktop.java
窗口主体程序。
2. Game.java
游戏核心算法程序。
3. GameJPanel.java
游戏显示画板程序.
4. GetKey.java
按键获取程序。
5. Point.java
基础点类。
